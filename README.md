# Portfolio Website

A full-stack portfolio website built with React.js frontend and Express.js/Node.js backend.

## Project Structure

```
Portfolio/
├── frontend/          # React.js application
├── backend/           # Express.js API server
└── README.md         # Project documentation
```

## Features

- Modern, responsive design
- Fast React.js frontend
- Express.js backend API
- Mobile-first responsive design
- Portfolio showcase
- Contact form functionality
- Skills and experience display

## Getting Started

### Frontend (React)

```bash
cd frontend
npm install
npm start
```

### Backend (Express)

```bash
cd backend
npm install
npm run dev
```

## Tech Stack

**Frontend:**

- React.js
- React Router
- Responsive Design

**Backend:**

- Node.js
- Express.js
- CORS middleware
- Environment variables

## Development

The frontend runs on `http://localhost:3000` and the backend API runs on `http://localhost:5000`.
