## Getting Started

### Frontend (React)

```bash
cd frontend
npm install
npm start
```

### Backend (Express)

```bash
cd backend
npm install
npm run dev
```

## Tech Stack

**Frontend:**

- React.js
- React Router
- CSS3/SCSS
- Responsive Design

**Backend:**

- Node.js
- Express.js
- CORS middleware
- Environment variables

## Development

The frontend runs on `http://localhost:3000` and the backend API runs on `http://localhost:5000`.
